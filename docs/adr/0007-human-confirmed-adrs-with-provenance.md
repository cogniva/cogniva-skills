# ADRs are human-confirmed and carry provenance + relitigation weight

**Provenance:** Required by human
**Relitigation:** Blockers only

ADRs are never written without explicit human confirmation, and each records its
provenance (Suggested by agent / Suggested by human / Required by human) plus an
optional relitigation weight, because silently auto-written ADRs produced a pile of
noisy, unwanted constraints on development.
