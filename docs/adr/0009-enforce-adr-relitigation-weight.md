# Relitigation weight is enforced by the skills, not merely recorded

**Provenance:** Suggested by human
**Relitigation:** Compelling reasons only

The feature-lifecycle skills read existing ADRs and honour each decision's
relitigation weight before reopening it — surfacing a needed change to the human with
a reason rather than quietly working around it or re-proposing it — so deliberate
guardrail decisions stop being relitigated by an agent that lacks the context for why
they exist.
